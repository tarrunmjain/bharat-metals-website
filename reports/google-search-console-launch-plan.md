After launch:
1. Add URL-prefix property: https://www.stainlesssteeldealers.com/
2. If possible, also create Domain property: stainlesssteeldealers.com
3. Verify ownership using DNS TXT if required.
4. Submit sitemap: https://www.stainlesssteeldealers.com/sitemap.xml
5. Use URL Inspection for priority URLs.
6. Request indexing for top money pages.
7. Monitor indexing status, crawl errors, duplicate canonical issues, mobile usability and rich results/schema errors.
8. Record date of sitemap submission.
9. Use reports/search-console-priority-submit-list.md for the top 75 priority URLs.

DNS TXT safety:
- Add only the TXT record Search Console provides.
- Do not remove MX, SPF, DKIM or DMARC records.
- Keep verification TXT record in place unless Search Console says otherwise.
